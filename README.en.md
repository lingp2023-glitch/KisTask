# KisTask

#### Description
An open-source project management software that supports multi-project management, applied to various forms of task management such as BUG tracking and requirement management.

#### Software Architecture
The system is designed with a separated front-end and back-end architecture,
The server-side is developed using PHP+MySQL,
The client-side uses the VUE architecture.

#### Installation

1.  Prepare the PHP runtime environment (php+mysql+redis) on the Web server.
2.  Upload the KisTask code to the Web service directory.
3.  Create a database and import the kis.sql data.
4.  Modify the config/config_db.php file, updating the MySQL database and Redis connection information.
